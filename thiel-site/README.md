# Thiel Fellowship Personal Site

This folder contains a static personal website for scholarship/application use.

## Quick Edit

Update the following fields in index.html before publishing:

- Name placeholder in the title
- Email
- GitHub profile
- LinkedIn profile

## Deploy to GitHub Pages

1. Create a repository (for example: your-username.github.io)
2. Upload files in this folder to the repository root
3. In GitHub repository settings, enable Pages from the default branch/root
4. Wait for deployment and open the provided Pages URL
